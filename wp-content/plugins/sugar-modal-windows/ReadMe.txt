================ INSTALLATION ================

To install Sugar Modal Windows, simply upload the unzipped folder called "sugar-modal-windows" to yout wp-content/plugins directory.

Next, navigate to your WordPress plugins menu and click "Activate" on "Sugar Modal Windows".

You will now have a new menu item called "Modals". This is where you will create your Modal Windows.

================ USAGE =======================

For your convienance, all usage documentation has been included inside WordPress.

Click on the "Help" link inside the "Modals" menu.
